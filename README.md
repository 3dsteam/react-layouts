# React Layouts

Page layouts for React

## Installation

```bash
  npm install @3dsteam/react-layouts
```

### Prerequisites

For this package you need to have a `.npmrc` file in your project root with the following content:

```text
@3dsteam:registry=https://npm.pkg.github.com
```
